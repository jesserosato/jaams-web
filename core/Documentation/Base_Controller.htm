JAAMSBase Class

 

Class Description

An “abstract” MVC controller class.

 

Class Properties

* indicates that a value for that property must be passed in the class constructor.

 

model

-       Type: Mixed

-       Default value: null

-       Expected values: Any class that accepts a JAAMSController object as the first parameter of its constructor.

-       Description: Data store interface.

view

-       Type: String

-       Default value: Empty string

-       Expected values: String of HTML.

-       Description: The HTML to display the object.

dir_paths*

-       Type: Array

-       Default value: Array with two empty elements: ‘view’ and ‘model’.

-       Expected values: An array containing two arrays, keyed to ‘view’ and ‘node’.

-       Description: Array of paths (absolute, or relative to the class definition file directory) to directories to search (in order) for view and model file(s).  See the public methods “get_view” and “get_model” for more information about how the class searches for view and model files.

hierarchies

-       Type: Array

-       Default value: Array with ‘view’ and ‘model’ elements.

-       Expected values: An array containing two arrays, keyed to ‘view’ and ‘node’.

-       Description: An array describing the filename structure for views and models.  Basically, the elements of ‘view’ and ‘model’ arrays each get imploded using the appropriate value in class property ‘seps’, the appropriate extension (in class property ‘exts’) is appended, and the appropriate directory paths are searched for that filename.  If none is found, the last element of the array is removed, and the process is repeated.

sets

-       Type: Array

-       Default value: Array with two empty elements: ‘view’ and ‘model’.

-       Expected values: An array containing two arrays, keyed to ‘view’ and ‘node’.

-       Description: The separators to use for filename parts (see ‘hierarchies’).

exts

-       Type: Array

-       Default value: Array with two empty elements: ‘view’ and ‘model’.

-       Expected values: An array containing two arrays, keyed to ‘view’ and ‘node’.

-       Description: The file extensions to use (see ‘hierarchies’).
