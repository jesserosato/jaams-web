JAAMSTemplatable Class

 

Class Description

A simple, yet general, base class for concepts that implement template(s).

 

Class Properties

* indicates that a value for that property must be passed in the class constructor.

 

template_dir_paths*

-       Type: Array

-       Default value: Array whose first element is a string describing the class definition file path.

-       Expected values: An array of strings describing a well-formed paths to readable directories.

-       Description: Array of paths (absolute, or relative to the class definition file directory) to directories to search (in order) for template file(s).  See the public method “get_template” for more information about how the class searches for template files.

 

template_hierarchy

-       Type: Array

-       Default value: array(‘template’, ‘form’);

-       Expected values: An array of strings that do not violate any system file naming conventions (i.e. no Unicode characters on systems that don’t support Unicode characters in filenames).

-       Description: A non-associative array describing the hierarchy to use when searching for a template file (see JAAMSTemplatable class).

template_ext

-       Type: String

-       Default value: ‘php’

-       Expected values: String containing a valid file extension.

-       Description: Template file extension.

template_sep

-       Type: String

-       Default value: ‘-’

-       Expected values: String containing a valid file extension.

-       Description: Template file extension.

 

 

 

Class Methods

get_template_path

-       Status: public

-       Parameters:

-       Returns:

o   String describing the path to the specified template file.

get_template

-       Status: public

-       Parameters:

-       Returns:

o   String output by the specified template file.

-       Example:

o   $obj = new JAAMSTemplatable();

o   $hierarchy = array(‘form’, ‘input’, ‘text’);

o   $ext = ‘php’;

o   $delim = ‘_’;

o   $html = $obj->get_template_file($hierarchy, $ext, $delim);

o   If ‘form_input_text.php’ exists, the output of that file will be stored in $html.  If ‘form_input_text.php’ is not found, this method will then look for ‘form_input.php’, followed by ‘form.php’.  If none of those are found, the method will throw an exception.
