Base Class

 

Class Description

“Abstract” base class for defining and generating HTML forms and form elements using templates.  See Form for usage examples.

 

Class Inheritance

Extends core/JAAMSTemplatable

 

Class Properties

name

-       Type: String

-       Default value: None

-       Expected values: Any string that can be used for an HTML tag attribute.

-       Description: HTML form tag “name” attribute.  Also used internally as a general handle for the form.

label

-       Type: String

-       Default value: Empty string

-       Expected values: Any string.

-       Description: A human readable name for the form.

errors

-       Type: Array

-       Default value: Empty string

-       Expected values: Array of errors or error messages.  Implementation is generic.

-       Description: An array of errors or error messages to use in templates.

 

Class Public Methods

__construct

-       Parameters

o   name (see Class Properties), required.

print_html

-       Convert the form to HTML using appropriate templates and output it.

get_html

-       Return: String

-       Convert the form to HTML and return that HTML as a string.

get_template_data

-       Return: Array

-       Package the object’s data for template use.  Returned array is keyed using the class property names.
